How to get it

Maven, Gradle...

Maven artifacts of Hibernate ORM are published to Maven Central and to the JBoss Maven Repository. Refer to the Maven Getting Started guide on the Jboss Wiki for more information on how to configure Maven.

Below are the Maven coordinates of the main artifacts.

org.hibernate:hibernate-core:4.3.11.Final
Core implementation
org.hibernate:hibernate-entitymanager:4.3.11.Final
JPA support
org.hibernate:hibernate-osgi:4.3.11.Final
OSGi integration
org.hibernate:hibernate-envers:4.3.11.Final
Envers audit support
org.hibernate:hibernate-c3p0:4.3.11.Final
c3p0 connection pooling
org.hibernate:hibernate-proxool:4.3.11.Final
Proxool connection pooling
org.hibernate:hibernate-infinispan:4.3.11.Final
Infinispan second-level caching
org.hibernate:hibernate-ehcache:4.3.11.Final
Ehcache second-level caching
