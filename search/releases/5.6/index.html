Hibernate Search 5.6 series stable

Introduces experimental support for Elasticsearch, last release supporting Hibernate ORM 5.0.z or 5.1.z
Compatibility Get it What's new Migrate Main documentation

Compatibility

Java
7
Hibernate ORM
5.0 - 5.1
Elasticsearch
2.0 - 2.4
Apache Lucene
5.5

Not compatible with your requirements? Have a look at the other series.

How to get it

Direct download is available from SourceForge.

Maven artifacts of Hibernate Search are published to Maven Central and to the JBoss Maven Repository. Refer to the Maven Getting Started guide on the Jboss Wiki for more information on how to configure Maven.

Below are the Maven coordinates of the main artifacts.

org.hibernate:hibernate-search-orm:5.6.3.Final
Hibernate ORM integration
org.hibernate:hibernate-search-elasticsearch:5.6.3.Final
Elasticsearch integration
org.hibernate:hibernate-search-backend-jgroups:5.6.3.Final
JGroups backend
org.hibernate:hibernate-search-backend-jms:5.6.3.Final
JMS backend

More information about specific releases (announcements, download links) can be found here.
