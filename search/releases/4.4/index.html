How to get it

Maven, Gradle...

Maven artifacts of Hibernate Search are published to Maven Central and to the JBoss Maven repository. Refer to the Maven Getting Started guide on the JBoss Wiki for more information on how to configure the JBoss Maven repository.

You can find the Maven coordinates of all artifacts through the link below:

Maven artifacts

Below are the Maven coordinates of the main artifacts.

org.hibernate:hibernate-search-orm:4.4.6.Final
Hibernate ORM integration
org.hibernate:hibernate-search-backend-jgroups:4.4.6.Final
JGroups backend
org.hibernate:hibernate-search-backend-jms:4.4.6.Final
JMS backend
org.hibernate:hibernate-search-analyzers:4.4.6.Final
Analyzer dependencies from Lucene/Solr

To verify signed Maven artifacts, head to this page.
